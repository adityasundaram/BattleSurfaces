Features

Our Application is built in Unity, leveraging the ARCore library from Google. The application is one of the initial few games targeting the Table-Top gaming platform. The game showcases some of the challenges with AR and is an attempt to add another dimension to your mobile gaming experience.

Table-Top Game

The app we have built is a game that allows players to experience Table top gaming at any place, any time.

Handheld Controls

The game provides intuitive handheld controls to navigate the AR world spawned in front of you, mixing with your reality.

Augmented Reality

The application brings the mobile game to your world, augmenting the world through your camera. Relive some of the intense action through Augmented reality bringing your favorite game to life.

Android

This game is currently only for the android platform, for higher android versions that support ARCore
